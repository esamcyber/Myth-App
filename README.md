This ia a Myth App that tests users on myths and they guess whether or not it's true or false.

-Features
Welcome screen
Myth questions
Feedback regarding the myths 
Score tracking 
Final results

-How to use app
Click the start button
Read the question shown and answer the question 
Select whether the myth is true or false
Feedback will then be shown after answering each question 
Click NEXT to move on to the next question 
View results at the end
